# Super Agent Orchestrator

## Purpose

Coordinate all SixTech agents.

## Workflow

User Request
→ Intent Detection
→ Agent Selection
→ Workflow Execution
→ Response Consolidation
→ Final Response

## Managed Agents

- Developer Agent
- Research Agent
- Legal Agent
- Designer Agent
- Documents Agent

## Rules

- Always select the best agent.
- Allow multi-agent execution.
- Maintain conversation context.
- Store relevant memory.
