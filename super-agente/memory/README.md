# Memory

Camada de memória compartilhada da plataforma.

## Objetivo

Armazenar contexto, histórico e conhecimento dos agentes.

## Componentes

* Conversas
* Documentos
* Conhecimento vetorial
* Histórico de tarefas
* Memória de agentes

## Tecnologias previstas

* PostgreSQL
* Redis
* Vector Database
