# Routing

Sistema de roteamento do SuperAgente.

## Fluxo

Usuário
→ SuperAgente
→ Classificação da intenção
→ Seleção do agente
→ Execução
→ Resposta

## Agentes

* Desenvolvimento
* Pesquisa
* Jurídico
* Design
* Documentação

## Regras

* Selecionar o melhor agente.
* Permitir múltiplos agentes.
* Consolidar respostas.
